# Catur Lawan AI (Chess vs AI)

Ini adalah game catur sederhana berbasis web di mana Anda bisa bermain melawan komputer (AI Stockfish).

## Fitur
- **Lawan AI Cerdas:** Menggunakan engine Stockfish via Web Worker.
- **Tingkat Kesulitan:** Anda bisa mengatur ELO AI (misal 400 - 3000).
- **Sistem Rating:** Menyimpan rating (ELO) Anda di browser (LocalStorage).
- **Responsif:** Tampilan papan menyesuaikan layar.

## Cara Deploy ke Github Pages
1. Buat repository baru di Github dengan nama `CaturLawanAI`.
2. Upload file `index.html`.
3. Masuk ke **Settings** > **Pages**.
4. Pilih Branch `main` dan folder `/ (root)`, lalu **Save**.
5. Akses di `https://widiopranoto.github.io/CaturLawanAI/`.
